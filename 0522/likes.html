LIKES & INTERESTS

夏天溶解在我們若無其事的一天

02-A // FANDOM

坑位與作品

這裡可以詳細寫下你目前深陷在哪些坑裡！不論是長篇大論的碎碎念、推坑心路歷程，還是最近才剛摔進去的新世界。有了大區塊的保護，你可以打一整段漂亮的散文，字再多都會碰到邊邊自動換行，看起來就像一本精緻的雜誌。

02-B // CHARACTER

推し ／ 喜愛角色

這裡是你的大本命告白牆！可以列出你在各大作品裡的單推、CP、或是吃什麼配對。大空間可以讓你把對他們的愛寫得清清楚楚，完全不受原本窄格子的限制，愛怎麼宣示主權就怎麼寫。

02-C // MUSIC

音樂與旋律

樂團、歌手、曲風，任何讓你創作時充滿動力的聲音。你可以列出你最愛的 Spotify 歌單名，或是你最近在循環播放的特定幾首神曲，把你的聽覺品味也分享給進來參觀的人。

02-D // GAMES

遊戲世界

Splatoon 3 / Pokemon / Zelda / 各類遊戲。這裡非常適合放你的遊戲代碼、花枝的武器配裝心得、或是寶可夢的愛將名單。

02-E // DISLIKE

雷點與避諱

這裡可以寫一些你的社交禁忌，或是創作上比較避諱的事情。用一種乾淨整潔的排版把它寫出來，既有禮貌，又能精準傳達你的交友原則。
